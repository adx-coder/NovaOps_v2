# UI Recording Checklist: NovaOps Marketing Ad
**Capture Settings:** 1920x1080 (HD), 60 FPS (if possible for smooth scroll), Dark Mode enabled.

---

### 1. Website Showcase
- [ ] **The "Hero" Interactive Terminal:** record yourself clicking different tabs (OOM, Traffic Surge, etc.) and let the text type out fully.
- [ ] **The Bento Grid Scroll:** A slow, steady scroll through the "Features" section (Agentic War Room, Governance Gate, etc.).
- [ ] **The Evaluator CTA:** Hover over the "Get Started" button at the bottom of the page.

### 2. Dashboard: Live Action
- [ ] **The Incident Feed:** Scroll through the list of recent incidents on the dashboard home.
- [ ] **The Log Stream:** Filter the logs by "Nova-Agent" and show the clean, real-time investigation logs flowing.
- [ ] **The War Room View:** Open a specific incident (like the OOM one) and scroll through the Agent findings (Log Analyst, Infra Specialist).
- [ ] **The Jury Verdict:** Highlight the "Jury & Gov" section where the disagreement or agreement check is logged.

### 3. The "Approval" CLI
- [ ] **The Nova Sonic Call:** Record a terminal window running `./trigger_live_outage.sh`. Specifically capture the part where Nova Sonic speaks and you type "approve". 

---
**Tips for Premium Look:**
- Use a clean browser window (hide bookmarks bar, no extra tabs).
- Use a steady mouse speed.
- If using Chrome, consider using "Device Mode" set to a high-DPI Responsive window for crisp text.
